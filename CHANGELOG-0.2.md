CHANGELOG for 0.2.x
===================

This changelog references the relevant changes (bug and security fixes) done
in 0.2 minor versions.

#### 0.2.0 (2015-08-27)
  * enhancement [Invoice] handling invoices with Invoice object (WaSa42)
  * enhancement [Invoice] templating for sending invoices via emails (WaSa42)
  * deletion [PayPal] deletion of useless methods: createPayment() & executePayment() (WaSa42)
  * enhancement [Item] handling items for invoices with Item object (WaSa42)
  * enhancement [Invoice] create an invoice directly from a payment: Paypal.createInvoiceFromPayment() (WaSa42)
  * enhancement [i18n] English & french translations for an invoice's email (WaSa42)
